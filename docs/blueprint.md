# **App Name**: Tempo Flow

## Core Features:

- Beat Visualization: A main display showing a visual animation of the beat along with subdivisions, which are black, eighth note triplets and sixteenth notes.
- Playback Controls: Intuitive playback controls for starting, stopping and looping the beat.
- Tempo Section List: Display tempo sections as colored stacked blocks, displaying parameters such as name, tempo, and time signature.
- Section Editor: Modal form for the creation and editing of tempo sections, including tempo, beat and subdivision options.
- Section Reordering: Allows a user to rearrange song sections with drag and drop
- Preset Management: The ability to save and load sequence presets, exporting and importing them to local and cloud storage

## Style Guidelines:

- Dark mode as the default theme, with light mode as an option.
- Each section has its own different identifiable color.
- Accent: Use Teal (#008080) to highlight interactive elements.
- Use clear, legible fonts, optimized for mobile screens.
- Simple and minimalist icons.
- Optimized touch controls with big buttons.
- Subtle and non-intrusive animations for feedback and transitions.
